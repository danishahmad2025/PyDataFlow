# PyDataFlow

PyDataFlow is a lightweight, end-to-end data engineering pipeline built using pure Python.

## Project Goals
- Learn real-world data engineering concepts
- Build scalable data ingestion pipelines
- Apply data validation and transformation
- Practice logging, error handling, and modular design
- Create a resume-ready open-source project

## Tech Stack
- Python
- Pandas
- NumPy
- SQLite
- Logging
